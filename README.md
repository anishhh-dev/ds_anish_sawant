## Project Overview
This project analyzes the relationship between trader performance and Bitcoin market sentiment.
The goal is to understand how trading behavior and profitability change during Fear and Greed
market conditions.

The analysis is part of a hiring assignment for a Web3 trading team.


## Datasets Used
1. Historical Trader Data  
   - Contains trade-level information such as profit/loss, leverage, and timestamps.

2. Bitcoin Fear & Greed Index  
   - Provides daily market sentiment classified as Fear or Greed.


## Objective
To explore how trader performance, risk-taking behavior, and trading activity vary under
different market sentiment conditions and to uncover insights that can support smarter
trading strategies.
